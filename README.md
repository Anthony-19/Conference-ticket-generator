# Conference Ticket Generator

A simple and interactive web application that allows users to register for a coding conference, upload their avatar, and generate a unique ticket ID. All user data, including the uploaded image, is stored in `localStorage` and can be accessed on a confirmation page.

---

## Features

- User-friendly registration form for coding conference attendees.
- Upload and preview an avatar image (supports JPG and PNG, max 500KB).
- Real-time form validation for:
  - Full Name
  - Email Address
  - GitHub Username
  - Avatar upload
- Generate a unique ticket ID in the format `#XXXXX`.
- Persist form data in `localStorage` to be accessed on a confirmation page.
- Change or remove the uploaded image before submission.
- Redirect to a ticket confirmation page upon successful submission.
